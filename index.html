Cursos Hechos

HTML5

Es un Lenguaje de Marcado de Hipertexto, un estándar que sirve para definir la estructura, el diseño y el contenido de una página web como imágenes, texto, vídeos, juegos, marcos, estilo de letra, etc.

CSS3

Permite definir y crear la presentación visual de un documento ya escrito. Es decir generar el diseño visual de páginas web e interfaces de usuario a un documento HTML para decorarlo.

JavaScript

Es un lenguaje de programación para hacer páginas web interactivas. Las funciones de JavaScript pueden mejorar la experiencia del usuario de un sitio web y agrega funciones adicionales.
